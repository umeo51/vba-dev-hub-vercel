# お問い合わせフォーム＆FAQ実装レポート

## 📋 概要

VBA Dev Hubのお問い合わせフォームをGoogleフォーム埋め込み式に変更し、充実したFAQページを作成しました。これにより、ユーザーサポート体制が整い、実運用が可能になりました。

**実装日**: 2025年11月22日  
**ステータス**: 実装完了・デプロイ済み

## ✨ 実装内容

### 1. お問い合わせフォームのGoogleフォーム化

#### 変更前
- フロントエンドのみのフォーム
- 送信機能なし（モックアップ）
- データの保存なし

#### 変更後
- **Googleフォーム埋め込み式**
- 実際の送信機能
- 回答の自動保存（Googleスプレッドシート）
- メール通知対応（設定可能）

#### 実装の特徴

**柔軟な設定方法**:
```typescript
const GOOGLE_FORM_URL = "https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true";
```

- URLを置き換えるだけで動作
- Googleフォーム未設定時は、設定手順を表示
- 設定済みの場合は、フォームを埋め込み表示

**ユーザー体験**:
- ページ内で完結（別タブ不要）
- レスポンシブデザイン
- 送信後の確認メッセージ

## 📝 Googleフォーム設定手順（要約）

### ステップ1: フォーム作成

1. https://forms.google.com/ にアクセス
2. 「空白」を選択
3. 以下の質問を追加:
   - お名前（記述式、必須）
   - メールアドレス（記述式、必須、検証あり）
   - 件名（プルダウン、必須）
   - お問い合わせ内容（段落、必須）

### ステップ2: デザイン設定

- テーマの色: 紫色（#9333EA）
- 確認メッセージ: カスタマイズ

### ステップ3: 埋め込みURL取得

1. 「送信」→「<>」（埋め込み）タブ
2. iframe内のURLをコピー

### ステップ4: サイトに設定

1. `app/contact/page.tsx`を開く
2. `GOOGLE_FORM_URL`にURLを貼り付け
3. 保存してデプロイ

詳細は`GOOGLE_FORM_SETUP.md`を参照。
