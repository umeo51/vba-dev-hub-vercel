# VBA Dev Hub

VBA開発者のための統合支援プラットフォーム

## 概要

VBA Dev Hubは、VBA（Visual Basic for Applications）を使用する開発者や学習者を支援するための総合的なWebアプリケーションです。日常的なコーディング作業の効率化から、知識の共有、スキルアップまで、VBAに関わるあらゆるニーズに応えます。

## 主な機能

### 1. コード生成・効率化ツール群

- **VBAコード自動生成ツール**: よくある定型処理のコードを、UIベースで簡単に生成
- **コード整形ツール**: インデントや命名規則を自動で統一し、コードの可読性を向上
- **正規表現テスター**: VBA用の正規表現パターンをリアルタイムでテスト

### 2. ナレッジ共有・検索システム

- **VBAスニペット共有**: ユーザーが便利なコード断片を投稿・検索・評価できるプラットフォーム
- **エラーコード逆引き辞典**: 頻出するエラーの原因と対策を網羅的に解説
- **逆引きリファレンス**: 「やりたいこと」から必要なコードを素早く検索
- **関数・ステートメント解説**: サンプルコード付きの詳細なリファレンス

### 3. 学習・コミュニティ機能

- **VBAクイズ・練習問題**: スキルアップのための学習コンテンツ
- **ユーザーフォームシミュレーター**: Web上でUIデザインとコード生成を体験

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript
- **バックエンド**: Python (Flask/Django)
- **データベース**: SQLite
- **ホスティング**: GitHub Pages / Vercel / Heroku (予定)
